# Hybrid Quantum-Classical GAN (QGAN) Project

هذا المشروع عبارة عن تطبيق لنموذج **Quantum GAN (QGAN)** الهجين، الذي يجمع بين الحوسبة الكمومية والتعلم العميق الكلاسيكي لتوليد بيانات جديدة تحاكي توزيعات معينة.

## هيكل المشروع

يتكون المشروع من الملفات التالية:

1.  **`environment_setup.py`**: ملف لاختبار بيئة العمل والتأكد من تثبيت المكتبات اللازمة (`PennyLane`, `PyTorch`, `Matplotlib`).
2.  **`quantum_generator.py`**: يحتوي على تعريف **المولد الكمومي (Quantum Generator)** باستخدام دوائر PennyLane المتشابكة بقوة.
3.  **`classical_discriminator.py`**: يحتوي على تعريف **المحقق الكلاسيكي (Classical Discriminator)** المبني باستخدام PyTorch.
4.  **`training_loop.py`**: الكود الرئيسي الذي يجمع المكونات ويقوم بعملية التدريب وتوليد النتائج.
5.  **`qgan_loss.png`**: رسم بياني يوضح تطور الخسارة (Loss) لكل من المولد والمحقق أثناء التدريب.
6.  **`qgan_generated_data.png`**: رسم بياني يقارن بين البيانات الحقيقية والبيانات المولدة بواسطة النظام الكمومي.

## كيفية التشغيل

1.  تأكد من تثبيت المكتبات المطلوبة:
    ```bash
    pip install pennylane torch matplotlib qiskit
    ```
2.  قم بتشغيل حلقة التدريب لرؤية النتائج:
    ```bash
    python training_loop.py
    ```

## المكونات التقنية

*   **المولد الكمومي**: يستخدم 4 كيوبرت (Qubits) مع طبقات تشابك قوية (`StronglyEntanglingLayers`) لخلق توزيعات احتمالية معقدة.
*   **المحقق الكلاسيكي**: شبكة عصبية بسيطة (MLP) تحاول التمييز بين البيانات الحقيقية والبيانات المولدة.
*   **التدريب**: يتم استخدام خوارزمية Adam لتحسين أداء كلا النموذجين في بيئة تنافسية.
